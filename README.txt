Sistema de Ventas XYZ

Este es un sistema de ventas desarrollado en Python para la empresa XYZ. El sistema cuenta con funcionalidades para gestionar clientes, productos, ventas e inventarios.

Estructura del Proyecto:
- configuraciones.py: Contiene funcionalidades para la configuración del sistema, como inicio de sesión, registro de usuarios y restablecimiento de contraseñas.
- clientes.py: Contiene funcionalidades relacionadas con la gestión de clientes, como registro de nuevos clientes, eliminación y actualización de datos.
- inventarios.py: Contiene funcionalidades para la gestión de inventarios, como creación de inventarios, agregado de productos y listado de inventarios.
- ventas.py: Contiene funcionalidades relacionadas con la gestión de ventas, como registro de ventas, visualización de historial y eliminación de ventas.
- main.py: Archivo principal que coordina la ejecución de los diferentes módulos y muestra el menú principal.

Instrucciones de Uso:
1. Ejecuta el archivo main.py con Python para iniciar el programa.
2. Selecciona una opción del menú principal para acceder a las diferentes funcionalidades del sistema.
3. Sigue las instrucciones en pantalla para interactuar con el sistema y realizar operaciones como registro de clientes, registro de ventas, etc.

Notas:
- Todos los datos se guardan en archivos dentro de la carpeta "Archivos".
- Las contraseñas de los usuarios se almacenan encriptadas en el archivo contraseñas.txt.

¡Disfruta utilizando el sistema de ventas XYZ!
